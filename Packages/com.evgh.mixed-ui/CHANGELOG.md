# Changelog

## 0.2.0 - 2026-07-10

- Rebuilt the package as an XRI world-space uGUI visual component kit.
- Added `MixedUITheme`, `MixedUIThemeScope`, and visual-only style bindings.
- Added semantic button variants, labels, toggle, slider, card, stack, modal, scroll panel, and XRI Canvas prefabs.
- Added an XRI `LazyFollow` Canvas and Visual Gallery sample.
- Removed contextual builders, commands, factory/provider APIs, domain targets, custom spatial placement, custom interaction states, and the Canvas adapter.
- Replaced wrapper behavior with standard uGUI events and `Selectable.ColorBlock` states.

## 0.1.0 - 2026-07-10

- Initial contextual runtime UI architecture.
